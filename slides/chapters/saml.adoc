=== SAML

* Security Assertion Markup Language
* XML-based standard for Single Sign-On
* Uses signed assertions instead of JWTs
* Supported by Keycloak

=== SAML Assertion

* Identifies who authenticated and when
* May include roles, groups, and attributes
* Similar job to an ID Token, using XML

[source,xml]
----
<Assertion>
  <Subject>admin@example.com</Subject>
  <Attribute Name="role">admin</Attribute>
</Assertion>
----

=== SAML and Keycloak

* Keycloak can provide SAML login
* Keycloak can trust an existing SAML Identity Provider
* External identities map to Keycloak users, roles, and claims
* Web Applications can use legacy and modern identity systems together
