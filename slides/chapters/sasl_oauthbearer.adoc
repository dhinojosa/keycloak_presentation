=== SASL

* Simple Authentication and Security Layer
* Pluggable authentication for application protocols
* Used by Kafka, LDAP, IMAP, SMTP, and XMPP
* Client and server negotiate a mechanism
* The protocol does not invent authentication from scratch

=== OAuthBearer over SASL

* OAuthBearer is a SASL mechanism
* Client gets an Access Token from Keycloak
* Client presents the token during SASL authentication
* Kafka validates the JWT with JWKS
* Kafka authorizes using roles, scopes, or claims

=== Kafka OAuthBearer Wiring

[cols="35b,65"]
|===
| Setting | Job

| `sasl.mechanism`
| Selects `OAUTHBEARER`

| `sasl.jaas.config`
| Selects Kafka's OAuthBearer login module

| `sasl.login.callback.handler.class`
| Fetches a token from Keycloak for the client

| `sasl.server.callback.handler.class`
| Validates incoming tokens on the broker

| `sasl.oauthbearer.*.url`
| Points Kafka at Keycloak token and JWKS endpoints
|===
