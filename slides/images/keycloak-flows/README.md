# Keycloak Flow SVG Assets

Standalone `1280 x 720` SVG sequences for Keycloak flow concepts. Include with
`image::keycloak-flows/<flow>/<frame>.svg[width=100%]`.

Each subdirectory is one flow:

- `oidc`
- `refresh-token`
- `client-credentials`
- `device-authorization`
- `jwt-jwks`
- `saml`
- `oauthbearer`
- `password-flow-caution`

The diagrams are static SVG frames. The animation effect comes from advancing
through successive Reveal.js slides.
